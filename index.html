Weather Forecasting Web App

I developed a responsive Weather Forecasting Web Application using HTML, CSS, and JavaScript, integrated with the OpenWeatherMap API to deliver real-time climate data. The application allows users to search for any city and instantly view live weather details such as temperature, humidity, wind speed, and general conditions. Designed with a user-friendly interface, it ensures smooth performance across all devices and incorporates dynamic content rendering, asynchronous data handling using JavaScript’s `fetch` method, and basic error handling for invalid inputs or network issues. This project showcases my skills in modern frontend development, API integration, and building interactive, data-driven web solutions.

Spare & Scrap Hub

I developed a dynamic Spare and Scrap Marketplace web application using HTML, CSS, JavaScript, and a backend database to facilitate the buying and selling of spare parts and scrap materials. The platform offers a user-friendly interface for listing, browsing, and managing products, with responsive design for seamless access across devices. JavaScript powers interactive features and real-time content updates, while the database ensures secure data storage for user accounts, product listings, and transaction records. The project emphasizes functionality, performance, and ease of use, making it a practical solution for individuals and businesses dealing with spare and scrap items. This showcases my full-stack development skills and experience in building real-world, data-driven applications.
